# pygtkdrawingwindow

## Requirements

- [`Python 2`](https://www.python.org/)
- [`pyGTK`](http://www.pygtk.org/)
- [`pycairo`](https://cairographics.org/pycairo/)
- (optional) [`python-rsvg`](http://ftp.gnome.org/pub/GNOME/sources/gnome-python-desktop/)

## Installation

```
python setup.py install
```
